web-crawler-nodejs
==================

This is NodeJS crawler that uses https://github.com/sylvinus/node-crawler.
<br />
Our goal is to crawl information about happy hour in Austin, TX.

Running Environmnent
--------------------
Node v0.10.18 <br />
node-crawler

To install node-crawler: <br />
<code>
	$ npm install crawler
</code>
<br />

To run crawler, do 
<br />
<code>
	$ node crawler
</code>
